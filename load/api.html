Introduction

Welcome to the CaptchaⱮaster API. Our RESTful API allows you to integrate AI-powered CAPTCHA solving into your applications seamlessly. You can solve reCAPTCHA, hCaptcha, Turnstile, and more in under 1 second.

Base URL:

https://api.captchamaster.io/v2

Authentication

All API requests require a Bearer token. Include your API key in the Authorization header.

Header Example
# Header
Authorization: Bearer YOUR_API_KEY
POST /solve

Submit a CAPTCHA solving task. Returns the solved token within the average response time of 1 second.

Parameters

Name Type Required Description
typestringYesCAPTCHA type: recaptcha_v2, recaptcha_v3, hcaptcha, turnstile
sitekeystringYesThe site key from the target page
urlstringYesURL of the page with the CAPTCHA
invisiblebooleanNoSet true for invisible CAPTCHAs (default: false)

Request Example

JSON Body

  "type": "recaptcha_v2",

  "url": "https://example.com/login",
  "invisible": false
}

Response Format

All API responses return JSON with the following base structure:

{
  "success": true | false,
  "request_id": "string",
  "solution": "string" | null,
  "error": "string" | null,
  "solve_time": 0.00
}

Error Codes

401

Unauthorized

Invalid or missing API key in the Authorization header.

402

Insufficient Balance

Your account does not have enough credits to process this request.

422

Invalid Parameters

Missing required fields or invalid CAPTCHA type specified.

429

Rate Limit Exceeded

Too many requests. Please retry after the time specified in headers.

500

Server Error

An internal error occurred. Please try again or contact support.

SDK Examples

Use our official SDKs to integrate faster. Available for Python, Node.js, and Go.

PYTHONExample
solve.py
import captchamaster

client = captchamaster.Client("YOUR_API_KEY")

result = client.solve(
  type="recaptcha_v2",
  sitekey="6Le-wvkS...",
  url="https://example.com"
)

print(result.solution) # Solved token
NODE.JSExample
solve.js
const CM = require('@captchamaster/sdk');
const client = new CM('YOUR_API_KEY');

const result = await client.solve({
  type: 'recaptcha_v2',
  sitekey: '6Le-wvkS...',
  url: 'https://example.com'
});

console.log(result.solution);
